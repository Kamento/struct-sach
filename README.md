# struct-sach

Chương trình viết bằng ngôn ngữ C sử dụng `struct` để quản lý thông tin sách.

## Nội dung chính
- Khai báo cấu trúc `SACH`
- Nhập và xuất thông tin 1 quyển sách
- Nhập và xuất danh sách n quyển sách
- Ghi và đọc danh sách sách từ file

## Ngôn ngữ sử dụng
- C (Dev-C++)

## Mục đích
Bài tập môn Kỹ thuật lập trình – ngành Công nghệ thông tin.
